Virtual Pet

Description

This was one of my first Java programs. The user is given a virtual pet, a hamster named Sweet Pea, to care for. They simply had to water and feed Sweet Pea and put it on the treadmill. However, by using a random number generator, the response to feeding, watering and exercising Sweet Pea is unpredictable. Taking care of pets is tricky business!

Virtual Pet Shelter

Description

This Java project builds on the previous virtual pet project. The user now has to care for multiple hamsters at the virtual pet shelter. New activities for the user were added such as adopting, admitting and playing with the pets. The attributes of the pets are changed based on the activity the user chooses. For example, when the pets are fed, hunger goes down.

Description
